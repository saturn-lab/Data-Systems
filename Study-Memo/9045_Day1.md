## GitHub和SQL实验
1、Git使用
Git是一套协议，GitHub把协议集成，形成一套工具

Git上项目的的操作步骤：
Repo Dir   建仓库
Add    加文件
Commit    Commit Change
Push Remote   各成员反馈意见
Merge     编辑汇总、合并等
Release   发行

2、github爬项目
1）安装Git软件

2)右键——Git Bash Here（爬到当前的文件夹下）——去复制url

当前文件夹下就会有一个项目
3)更新项目
到项目的文件夹里——执行git pull命令
注：一定要到项目文件夹下执行git pull，不然是不成功的


修改文件后可以用git push推送回去
