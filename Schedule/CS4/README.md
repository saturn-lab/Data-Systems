# Postgres 和 MySQL 比较


Postgres 和 MySQL 都有基于页面的物理存储。(8KB vs 16KB)

MySQL存储引擎MyISAM, InnoDB

MySQL： InnoDB 表结构 聚簇索引(Clustered Index)	

PostgreSQL： 堆上元组（HOT, Heap Only Tuple）
